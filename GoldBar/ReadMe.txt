You need to download the missing fonts from Google Fonts.

1. Visit https://fonts.google.com/
2. Search for the following fonts:
	Righteous; Bungee, Comfortaa
3. Download the fonts
4. Place the fonts in C:\PathToOpenScad\fonts
5. Each font has to be in its own Folder like so
	C:\PathToOpenScad\fonts\Bungee\Bungee-Regular.ttf
